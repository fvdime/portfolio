## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/fvdime/portfolio.git
    ```
   
2. Install NPM packages:
   ```bash
   npm install
    ``` 
   
3. Add .env file:
   ```bash
   RESEND_API_KEY="<RESEND API KEY>"
    ```

4. Run the development server:

   ```bash
   npm run dev
   ```
